# html.lang

- **Type:** `string`
- **Default:** `en`

Configure the HTML language attribute.

```html
<html lang="en"></html>
```

## Example

For example to set HTML language to `zh-CN`:

```js
export default {
  html: {
    lang: 'zn-CN',
  },
};
```

Result:

```html
<html lang="zn-CN"></html>
```
